## 📋 How to use this app locally

### 1. Clone the repository

```bash
git clone https://github.com/mrHock89/Product-Landing-Page-.git
cd Product-Landing-Page-
```
### 2. Open the App 

####  **🐧 ON LINUX**
```bash
xdg-open index.html
```

#### **ON MacOS**
```bash
open index.html 
```
#### **ON Windows**
```bash
start index.html 
```
